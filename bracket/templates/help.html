Leaderboard

In each round, you will get points if you pick the correct series winner.
When the number of games is correct as well, bonus points will be awarded.
Points will be given according to following table:

Points Bonus
Round 1 10 5
Round 2 20 10
Conference Finals 40 20
Finals 80 40

Users and Groups

You can create only one bracket per user account.
With this account, it is possible to join multiple public groups and compete with your bracket.

Create a new group

Project is still in beta. It is not yet possible to create a new group yourself.
All current groups are public and anyone can join.
If you wish to create a new group, leave a message on my GitHub repository.
